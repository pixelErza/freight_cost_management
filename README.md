## Project Overview
This project builds an end-to-end machine learning system to analyze vendor invoice data and support financial decision-making.

It focuses on:
- Predicting expected freight cost  
- Identifying and flagging high-risk invoices  

This project was developed as a personal/academic project using real-world styled data.

---

## Business Objectives
- Improve freight cost estimation accuracy  
- Detect anomalies and suspicious invoices  
- Reduce manual effort for finance teams  
- Enable faster decision-making  

**Target Users:**  
- Finance teams  
- Procurement teams  
- Audit teams
